# WaCraft-Sample

**Przykładowa strona firmy budowlanej** nowoczesny front, portfolio realizacji, kontakt z klientem i panel administracyjny w jednej aplikacji.

Projekt powstał jako **demo / sample** marki *WaCraft*: pokazuje, jak może wyglądać witryna wykonawcy remontów i wykończeń wnętrz, od pierwszego wrażenia na stronie głównej po zarządzanie treścią z poziomu admina.

---

## O projekcie

WaCraft-Sample to responsywna aplikacja webowa dla branży budowlanej. Łączy prezentację usług z praktycznym zapleczem: właściciel strony może dodawać projekty, przeglądać wiadomości z formularza i śledzić ruch na stronie bez grzebania w kodzie.

Strona publiczna jest po polsku, z animacjami i układem nastawionym na mobile. Panel admin jest osobną strefą z logowaniem.

---

## Strona publiczna

- **Hero** — kinowy baner z animacjami scroll / parallax
- **Portfolio** — realizacje w układzie parallax (wnętrza, remonty, wykończenia)
- **Kontakt** — formularz wiadomości, dane firmy, mapa OpenStreetMap
- **SEO** — meta tagi, structured data (LocalBusiness)

---

## Panel administracyjny

- **Projekty** — dodawanie, edycja i usuwanie realizacji widocznych na stronie
- **Wiadomości** — skrzynka z formularza kontaktowego (status, odczytane / nowe)
- **Analityka** — odwiedziny, źródła ruchu, urządzenia, wykresy w czasie
- **Zabezpieczenia** — usuwanie projektu wymaga dodatkowego potwierdzenia (fraza znana tylko właścicielowi strony)

Logowanie demo: `/login`

---

## Technologie

React · Vite · Tailwind CSS · shadcn/ui · Framer Motion · React Router · Supabase · Recharts

## Cel repozytorium

Repozytorium służy jako **wizytówka projektu** i baza kodu strony sample. Konfiguracja bazy danych i klucze API pozostają po stronie właściciela, nie są częścią publicznego opisu projektu.
